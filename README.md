
# INFE Greece Eurovision Poll 2026 - Plesk Guide

This app is optimized for **Static Hosting** so you can host it for free on your existing **Plesk** server without needing a Google billing account.

## 📍 Where is the Terminal?
If you don't see the black box at the bottom:
1. Move your mouse to the **very bottom edge** of this window.
2. Look for a thin grey bar or the words **Logs, Terminal, Console**.
3. **Click and drag that bar upwards** to reveal the panel.
4. Click the tab labeled **"Terminal"**.

## 🚀 How to get your HTML files
1. Click inside the black Terminal box.
2. Type exactly: `npm run build`
3. Press the **Enter** key.
4. Wait about 60 seconds. A new folder named **`out`** will appear in the file list on the **left side** of this screen.
5. Right-click the **`out`** folder and select **"Download"**.
6. **Upload to Plesk**: Log in to your Plesk panel, go to **File Manager** -> `httpdocs`, and upload the **contents** of your downloaded `out` folder.

## ✨ Features
- **Alphabetical Sorting**: All countries are automatically sorted A-Z.
- **Official Branding**: Official INFE Greece logo and colors.
- **100% Free**: No credit card or billing account required when hosted on Plesk.
