# **App Name**: EuroVisionary

## Core Features:

- Entry Browsing & Video Playback: View Eurovision entries categorized by year and decade. Each entry includes essential details (country, artist, song title) and an embedded video player for the performance.
- User Account & Secure Voting: Users can register and log in to cast votes (e.g., 1-12 points) for entries from a specific year. The system ensures only one vote submission per user per year.
- Dynamic Scoreboard: Display real-time aggregated voting results and average scores for entries, with options to sort and filter by year or decade.
- AI Vote Insight Tool: An AI tool that suggests potential reasons or provides descriptive text suggestions for why a user might have voted for or against a particular entry, enhancing user feedback.
- Admin Entry Management: An administrative interface for adding, editing, or removing Eurovision song entries, including details like country, artist, song title, year, and video URL.
- Web Hosting: The application will be hosted as a web app on the domain escnights.infegreece.com.

## Style Guidelines:

- A dark color scheme evokes the glamour and stage presence of Eurovision. The primary color is a vibrant electric violet (#5A3DF5), chosen for its energetic and sophisticated feel that pops against a dark backdrop. The background color is a deep indigo (#252637), offering a rich and subdued base. The accent color is a bright, clear cyan (#59C9F0), providing strong visual contrast for interactive elements and highlights.
- Headline font: 'Space Grotesk' (proportional sans-serif) for a modern, slightly futuristic feel, suitable for highlighting contest entries and section titles. Body text font: 'Inter' (grotesque-style sans-serif) for clear readability of song details, voting instructions, and descriptions across the application.
- Utilize sleek, outlined icons that reflect elements of music, competition, and international representation (e.g., star ratings, microphone, flag placeholders, trophy icons, voting checkmarks). Icons should maintain a consistent minimal aesthetic.
- A responsive, grid-based layout prioritizes entry visuals and embedded video players. Clear navigation elements (tabs or dropdowns) for selecting years and decades ensure easy access to content. Voting controls are intuitively placed next to each entry.
- Subtle, smooth animations for user interactions such as casting a vote, filtering through entries, or updating the scoreboard, providing engaging and fluid transitions without distracting from the content.
